## TypeScript 俳句

用类型系统写一首俳句：

```typescript
type 春 = {
  風: string;
  光: string[];
  静寂: boolean;
};

type 夏 = Pick<春, '風'> & {
  蝉: number;
};
```

### 代码与诗的共通之处

- **简洁** — 最少的字符表达最多的含义
- **节奏** — 换行和缩进形成视觉韵律
- **结构** — 类型系统就是诗歌的格律

> 优雅的代码，读起来像一首诗。

写代码和写诗一样，都是**在约束中寻找自由**。
